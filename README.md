# nvim配置

## 环境要求

nvim版本号：v0.8以上

## 使用方式

1、将本仓库克隆到~/.config/nvim文件夹下


```bash
    
    git clone git@github.com:ljtnono/nvim_config.git

```

2、使用vim打开文件，并安装相关插件

```bash

vim ~/.config/nvim/init.lua

# 打开文件之后执行命令

:PackerSync

```


## 功能列表

* 文件导航
* vim主题
* 状态栏
* lsp
* 自动补全
* tab栏切换
* bufferline
